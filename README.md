# homebrew-tap

Homebrew tap for [glebis](https://github.com/glebis)' projects.

## Casks

```bash
brew install --cask glebis/tap/cenno
```

| Cask | Description |
|------|-------------|
| [cenno](https://github.com/glebis/cenno) | Minimal floating panels that let MCP agents ask you questions |

## Formulae

```bash
brew install glebis/tap/transcriber-kit
```

| Formula | Description |
|---------|-------------|
| transcriber-kit | TranscriberKit |
